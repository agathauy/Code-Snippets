## Type any one of the following command to find os name and version in Linux:
```

cat /etc/os-release
lsb_release -a
hostnamectl

```

## Type the following command to find Linux kernel version:
```

uname -r

```

```

cat /proc/version

```

```

cat /etc/issue
more /etc/issue
less /etc/issue

```

```

man hostnamectl
man uname
man cat

```
